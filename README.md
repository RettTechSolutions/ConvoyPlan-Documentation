# ConvoyPlan – Dokumentation

Willkommen bei **ConvoyPlan**, der Plattform für die Planung, Koordination und Echtzeitverfolgung von Marschverbänden und Konvois im BOS-Umfeld.

---

## Was ist ConvoyPlan?

ConvoyPlan ist eine browserbasierte Anwendung für Einsatzorganisationen (THW, Feuerwehr, Rettungsdienst u. a.), die:

- **Routen interaktiv plant** – Startpunkt, Ziel, Wegpunkte und Halte auf einer Karte setzen
- **Zeitplanung automatisch berechnet** – Ankunfts- und Abfahrtszeiten je Wegpunkt basierend auf Marschgeschwindigkeit
- **Fahrzeuge verwaltet** – Funkrufnamen, Abmessungen, Kraftstoffdaten, Formationsposition
- **Live-Tracking ermöglicht** – Fahrer senden ihre GPS-Position, Planer sehen alle Fahrzeuge in Echtzeit
- **Offizielle Marschbefehle erzeugt** – PDF im Standardformat (Lage, Auftrag, Marschform, Kanalwechsel u. v. m.)
- **Öffentliche Freigabelinks** bereitstellt – Lageübersicht ohne Login für Beobachter

---

## Schnelleinstieg

| Schritt | Beschreibung |
|--------|-------------|
| 1. Konto erstellen | Registrierung mit E-Mail und Passwort |
| 2. Organisation anlegen | Team oder Einheit anlegen, Mitglieder einladen |
| 3. Konvoi erstellen | Name, Startzeit, Geschwindigkeiten festlegen |
| 4. Route planen | Start/Ziel auf Karte setzen, Wegpunkte hinzufügen |
| 5. Fahrzeuge zuweisen | Fahrzeuge erstellen und dem Konvoi zuordnen |
| 6. Marschbefehl exportieren | PDF, GPX oder JSON herunterladen |
| 7. Einsatz starten | Fahrer aktivieren Tracking, Planer beobachten live |
